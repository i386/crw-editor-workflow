# Source Hardening Pass

Use this workflow to improve sourcing on an existing page.

## Skills

- `crw-source-auditor`
- `crw-style-lint`
- `crw-ai-policy-finalizer`
- `crw-edit-packager`

## Steps

1. Fetch or paste the current page.
2. Split factual claims into a claim map.
3. Map each claim to its citation.
4. Check whether the source supports the claim.
5. Flag unsupported, weak, anonymous, stale, or inaccessible sources.
6. Suggest stronger sources or weaker wording.
7. Add access date/archive recommendations where useful.
8. Package the edit.

## Output

- Claim-source table.
- Unsupported claim list.
- Proposed citation or wording fixes.
- Suggested edit summary.
