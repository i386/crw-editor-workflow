# CRW Edit Packager

Use this skill at the end of any CRW workflow.

## Required Sections

- Page title.
- Page type.
- Workflow used.
- Proposed wikitext or patch.
- Source list and source-support notes.
- Unsupported or weak claims.
- Cargo field changes.
- Formatting and tone risks.
- Suggested edit summary.
- Human review checklist.

## Standards

- Be explicit about uncertainty.
- Separate verified facts from suggested prose.
- Keep edit summaries concise.
- If publishing risk is high, say so clearly.
- If source checking was not done, mark the bundle as not ready.

## Output Template

Use `templates/edit-bundle.md` as the structure when producing a full package.
